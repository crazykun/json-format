
# JSON 在线格式化工具

一个现代化、功能丰富的 JSON 在线格式化和验证工具。

### 🎯 核心功能
- **JSON 格式化** - 美化和格式化 JSON 数据

## 📖 使用说明

### 基本操作

1. **格式化 JSON**
   - 在左侧输入框中粘贴或输入 JSON 数据
   - 右侧会自动显示格式化后的结果
   - 点击"格式化"按钮重新处理

2. **压缩 JSON**
   - 点击"压缩"按钮移除所有空格和换行
   - 再次点击恢复格式化显示

3. **XML 转换**
   - 输入 XML 数据会自动转换为 JSON
   - 点击"转XML"按钮将 JSON 转换为 XML

4. **文件操作**
   - 点击"保存"按钮下载格式化后的文件
   - 点击"复制"按钮复制到剪贴板
   - 拖拽 JSON/XML 文件到输入框直接加载

### 快捷键

- `Ctrl/Cmd + S` - 保存文件
- `Ctrl/Cmd + K` - 清空内容
- `Ctrl/Cmd + C` - 复制结果（焦点在输出区域时）

### 高级功能

- **行号显示** - 点击"行号"按钮显示/隐藏行号
- **节点折叠** - 点击"折叠"按钮折叠所有 JSON 节点
- **错误定位** - 语法错误会高亮显示具体位置
- **状态监控** - 实时显示输入和输出状态
